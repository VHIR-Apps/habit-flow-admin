Push Notifications

Send notifications to all app users & force config refresh

How Push Notifications Work

1. You compose a notification here and click "Send".
2. The notification is saved to Firestore (admin/push_notification).
3. When any user opens the app, it checks for new notifications.
4. If there's a new notification (newer than last seen), it shows as a local notification.
5. Cost: ~1 Firestore read per user per app open = practically FREE.
6. No Cloud Functions or server needed!
